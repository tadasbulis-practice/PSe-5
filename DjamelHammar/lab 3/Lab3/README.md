LAB-3 – Interfaces and Polymorphism


OOP Laboratory Work Collection

Student: Djamel Hammar
Group: Pse-05
Course: Object-Oriented Programming (C#)

Goal

The goal of this laboratory work is to implement interfaces and polymorphism in object-oriented programming.



The project supports multiple interchangeable implementations:

* SimpleMenu / FancyMenu
* ConsoleStudentPrinter / DetailedStudentPrinter
* SimpleAverage / WeightedAverage
* LinearSearch / CaseInsensitiveSearch
* BasicValidator / StrictValidator

Benefits of the Architecture

* loose coupling,
* reusable code,
* easier maintenance,
* scalability,
* clean project organization.

Conclusion

This laboratory work demonstrates how interfaces and polymorphism improve flexibility and maintainability in software development.
