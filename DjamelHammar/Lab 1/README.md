LAB-1 – Basic Console Application

Goal

The purpose of this laboratory work is to create a simple console application using the principles of object-oriented programming in C#. The application allows users to manage a student group through an interactive menu system.

Description

The program provides a basic text-based menu where the user can:

* add new students,
* display all students,
* exit the application.

The project introduces the fundamental concepts of:

* classes,
* objects,
* methods,
* collections,
* console interaction.

Main Features

* Interactive console menu
* Student object creation
* Student list management
* Displaying student information
* Basic user input handling

Technologies Used

* C#
* .NET Console Application
* Object-Oriented Programming principles
