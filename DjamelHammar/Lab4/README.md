## Goal
The goal of this lab is to apply isolation in object-oriented programming by separating responsibilities into independent interfaces and implementations.



Description

The project demonstrates how application components can work independently from each other. Every module has a single responsibility and can be modified or replaced without affecting the rest of the system.

Isolation Concept

Isolation means that:

* menu logic is separated from business logic,
* validation is separated from data processing,
* printing is separated from calculations,
* searching is separated from storage.

This approach improves maintainability and readability.
